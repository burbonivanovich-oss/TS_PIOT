# CPA-продукты Контура

Партнёрская программа Контура — кабинет партнёра на referral.kontur.ru.
Вознаграждение — процент от оплаченных счетов приведённых клиентов.

Поле **erid** заполняется вручную перед публикацией каждого размещения
(требование закона о рекламе).

## Форматы размещения

### Врезка
Блок внутри статьи через `src/data/cpa-banners.ts`.
Структура записи:

```ts
'ключ': {
  id: 'ключ',
  eyebrow: 'Надзаголовок (продукт или категория)',
  title: 'Оффер из таблицы выше',
  description: 'Одно предложение — конкретная польза для читателя.',
  cta: 'Текст кнопки →',
  ctaHref: 'реферальная ссылка с erid',
  visual: { abbrev: 'АБВ', bg: '#цвет' },
},
```

Баннер подключается к статье через frontmatter:
```yaml
cpa: ключ
```
или автоматически по категории через `CATEGORY_DEFAULT_CPA` в том же файле.

### Баннер
Готовое изображение из кабинета партнёра (инструмент «Подготовить баннер»).
Доступен для: Маркировка, Эльба, Дополнительные услуги УЦ.
Размещается через `<img>` с реферальной ссылкой и атрибутом `data-pagefind-ignore`.

### Ссылка
Текстовая ссылка внутри тела статьи по месту. Используется для продуктов,
слабо связанных с основными темами сайта (EDI, Закупки, Толк).

## Обязательные требования к рекламным размещениям

- Каждое размещение требует **erid** — уникального идентификатора рекламы
  по закону о рекламе (ФЗ-38, ст. 18.1).
- erid получается в кабинете ОРД (оператора рекламных данных) перед публикацией.
- erid вписывается в реферальную ссылку или в текст пометки «Реклама».
- Без erid размещение незаконно.
