Онлайн-касса — это кассовый аппарат с фискальным накопителем (ФН), который
записывает данные о каждом расчёте и передаёт их в ФНС через оператора фискальных
данных (ОФД). Применение ККТ обязательно для большинства видов торговли и услуг
по Федеральному закону 54-ФЗ.

Для торговли маркированными товарами с декабря 2025 года касса дополнительно
обязана поддерживать ТС ПИоТ и работать по формату ФФД 1.2 с фискальным
накопителем ФН-1.2.

Выбор кассы зависит от сценария работы: смарт-терминал подходит для небольшой
розницы и общепита, фискальный регистратор — для рабочего места кассира с
отдельным компьютером. Для курьерской доставки и выездной торговли удобен
автономный кассовый аппарат.

В этом разделе — инструкции по выбору, регистрации, замене ФН, сравнение моделей
и разбор типичных ошибок при работе с кассой.
