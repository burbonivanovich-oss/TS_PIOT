# Этикетка — ЭДО и УПД для маркированных товаров: как правильно оформить документы

## Telegram

Маркированный товар на вашем складе может числиться чужим в «Честном знаке» — если универсальный передаточный документ (УПД) от поставщика не подтверждён. Продажа через кассу заблокируется.

Как устроена цепочка:
— При приёмке поставщик передаёт УПД с кодами маркировки через ЭДО
— Вы подтверждаете приёмку электронной подписью (УКЭП)
— Только после этого коды переходят на ваш ИНН в «Честном знаке»
— При продаже ТС ПИоТ проверяет: ваш ли этот код
— Неподтверждённый УПД = основание для штрафа по ст. 15.12 КоАП РФ

Срок подтверждения: молоко, вода, пиво, табак — 3 рабочих дня. Лекарства — 1 рабочий день.

**Заголовок Дзен:** Поставщик привёз партию обуви, склад принял, касса не пробивает: история про незакрытый УПД в «Честном знаке»

Магазин обуви в Краснодаре получил поставку из Москвы — 80 пар на 2,4 млн ₽. Кладовщик расписался в накладной, разместил товар в зале. Через два дня кассир пробивает первую пару — касса выдаёт ошибку «Код не принадлежит продавцу». Поставщик прислал УПД через ЭДО, но магазин не подтвердил его в течение 3 рабочих дней. Партия в «Честном знаке» всё ещё числится за продавцом.

В публичной проверке такой случай стоит штрафа по ст. 15.12 КоАП РФ — 50 000–300 000 ₽ для юрлица за реализацию немаркированного товара.

### Зачем нужен ЭДО при маркировке

С 01.01.2022 поставки маркированных товаров оформляются только электронным УПД через аккредитованного оператора ЭДО. Бумажная накладная больше не имеет юридической силы по этим товарам.

В «Честном знаке» каждый код DataMatrix привязан к конкретному ИНН — производителю, дистрибьютору, рознице. Передача товара между участниками должна сопровождаться переходом кода с одного ИНН на другой. Бумагой это невозможно — нужен документ с УКЭП обеих сторон, который автоматически передаёт сведения в ЦРПТ.

### Как работает цепочка приёмки

Поставщик формирует УПД в учётной системе и отправляет через оператора (Диадок, СБИС, Тензор). Если у покупателя другой оператор — работает роуминг, без настроек.

УПД приходит в ЛК покупателя. Ответственный сотрудник сверяет наименования, количество, коды с фактической поставкой и подписывает своей УКЭП. Оператор автоматически отправляет сведения в «Честный знак», коды переходят на ИНН покупателя.

Только после этого товар можно продавать через кассу с ТС ПИоТ — модуль при пробитии чека проверяет, принадлежит ли код продавцу. Если код «чужой» — продажа блокируется.

### Сроки подтверждения по товарным группам

— Лекарственные средства — 1 рабочий день.
— Молочная продукция, упакованная вода, пиво, табак — 3 рабочих дня.
— Одежда, обувь, шины, парфюмерия, фотоаппараты — 3 рабочих дня.

Если не подтвердить вовремя — штрафа «за неподтверждение» не выпишут, но партия физически не сможет продаваться через кассу. Товар на полках есть, продавать нельзя.

### Расхождения и что делать

Иногда фактическая поставка не совпадает с УПД: коробки не приехали, в упаковке другие модели, повреждены штрихкоды. В этих случаях покупатель не подписывает УПД, а отправляет уведомление об уточнении поставщику через ЭДО.

Поставщик корректирует УПД и отправляет новый со скорректированным списком кодов. Альтернативно — оформляется частичная приёмка с пометкой о реально полученных кодах.

Главное правило: не подписывайте УПД с кодами на товар, который вы не получили физически. Иначе коды повиснут на вашем ИНН без реального товара, и при инвентаризации придётся объяснять, куда они делись.

Реализация немаркированного товара по ст. 15.12 КоАП РФ: для должностных лиц 5 000–10 000 ₽, для юрлица 50 000–300 000 ₽ с конфискацией. Хранение для целей сбыта — те же размеры.

### Что запомнить

— Маркированные товары принимаются только по электронному УПД через аккредитованного оператора ЭДО. Бумажная накладная по ним недействительна.
— Подтверждать УПД нужно в течение 1 рабочего дня (лекарства) или 3 рабочих дней (молоко, вода, пиво, табак, одежда, обувь).
— До подтверждения коды числятся за поставщиком — касса с ТС ПИоТ блокирует продажу.
— При расхождениях отправляйте уведомление об уточнении, а не подписывайте УПД «как есть».
— Штраф по ст. 15.12 КоАП РФ за реализацию немаркированного товара — до 300 000 ₽ для юрлица с конфискацией.
— Если у сторон разные операторы ЭДО — работает роуминг, дополнительных настроек не требует.
