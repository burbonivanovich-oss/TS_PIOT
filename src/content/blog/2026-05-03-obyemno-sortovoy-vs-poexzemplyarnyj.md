При работе с маркированными товарами участники оборота используют два метода учёта: объёмно-сортовой и поэкземплярный. Выбор метода определяется нормативными документами для каждой товарной категории и влияет на то, как именно нужно принимать, передавать и продавать товар.

## В чём принципиальное отличие

**Объёмно-сортовой учёт (ОСУ)** — метод, при котором в электронном УПД указывается наименование товара и количество единиц в партии без перечисления кодов каждой упаковки. Коды DataMatrix хранятся в системе «Честный знак», но при передаче партии от продавца к покупателю передаётся только суммарное количество по позиции.

**Поэкземплярный учёт** — метод, при котором в электронном УПД перечислен каждый уникальный код DataMatrix. При передаче 1 000 упаковок в УПД содержится 1 000 кодов. Каждая единица прослеживается индивидуально — от производства до продажи.

## Таблица сравнения

| Параметр | Объёмно-сортовой | Поэкземплярный |
|---|---|---|
| Отражение в УПД | Количество по позиции | Код каждой единицы |
| Сканирование при приёмке | Не обязательно | Каждая упаковка |
| Сканирование при продаже | Зависит от категории | Каждая упаковка |
| Трудоёмкость | Ниже | Выше |
| Точность прослеживаемости | Партионная | Поединичная |

## Какие категории используют каждый метод

Категория и метод учёта определяются постановлением Правительства для каждой группы товаров:

**Объёмно-сортовой учёт применяется для:**
- Молочной продукции (молоко, кефир, йогурт).
- Мороженого.
- Питьевой воды.
- Безалкогольных напитков.
- Пива и пивных напитков.
- Снеков, соусов, специй (с 1 сентября 2026).
- Консервов (с 1 октября 2026).

**Поэкземплярный учёт применяется для:**
- Одежды и лёгкой промышленности (текстиль, кожаные изделия).
- Обуви.
- Шин и покрышек.
- Духов и туалетной воды.
- Игрушек (с 1 сентября 2026).
- Фотоаппаратов.

Для ряда категорий возможна комбинация: объёмно-сортовой в оптовом звене и поэкземплярный в рознице.

## Почему это важно для магазина

**При объёмно-сортовом учёте** магазин принимает товар по накладной с указанием количества. Но при продаже всё равно обязан сканировать каждую упаковку — это требование разрешительного режима. Разница в том, что в УПД при приёмке не нужно сверять каждый код — достаточно подтвердить количество.

**При поэкземплярном учёте** каждая упаковка сканируется дважды: при приёмке (сверка с УПД) и при продаже (вывод из оборота). Это увеличивает трудоёмкость, но обеспечивает полную прослеживаемость.

## Вывод из оборота через кассу

Независимо от метода учёта, при продаже через кассу с разрешительным режимом сканируется каждая единица товара. Подробнее о том, как это работает на кассе, — в статье [вывод товаров из оборота через кассу](/blog/2026-05-03-vyvod-iz-oborota-cherez-kassu/).

Разрешительный режим требует, чтобы касса была оснащена ТС ПИоТ и поддерживала ФФД 1.2. Подробнее — в материале [что такое ТС ПИоТ](/blog/2026-01-15-chto-takoe-ts-piot/).

## Особенности работы с ЭДО

При объёмно-сортовом учёте оператор ЭДО формирует УПД с указанием наименования товара, кода ТН ВЭД и количества. При поэкземплярном — дополнительно XML-файл с перечнем кодов, который передаётся через API «Честного знака».

Порядок работы с ЭДО при маркировке описан в статье [ЭДО и УПД для маркировки](/blog/2026-05-01-edo-upd-markirovka/).

## Переход между методами

Для одной категории товаров метод учёта не меняется по желанию предпринимателя — он установлен нормативно. Однако «Честный знак» периодически вносит изменения: например, молочная продукция исходно предполагала поэкземплярный учёт для товаров со сроком годности более 40 дней, а затем условия были скорректированы.

Следите за актуальными правилами на сайте честныйзнак.рф для вашей категории.

## FAQ

**Что будет, если выбрать не тот метод учёта?**
Выбора нет — метод определён нормативно. При нарушении порядка передачи данных (например, отправка УПД без кодов там, где требуется поэкземплярный учёт) система «Честного знака» не примет документ.

**Можно ли использовать ОСУ для товаров, требующих поэкземплярного учёта?**
Нет. Выбор метода — требование законодательства, а не технической возможности оператора ЭДО.

**Как узнать, какой метод применяется к конкретному товару?**
На сайте честныйзнак.рф в разделе «Документы» содержатся постановления и методические рекомендации по каждой категории.

**Нужно ли сканировать каждый товар при инвентаризации?**
Инвентаризация с маркированными товарами предполагает проверку соответствия физических остатков данным «Честного знака». При поэкземплярном учёте — сканирование каждой единицы. При ОСУ — сверка суммарного количества.

**Что происходит при несоответствии остатков в системе и на складе?**
Несоответствие нужно урегулировать через личный кабинет «Честного знака». Причинами могут быть: непроведённый возврат, ошибка при приёмке, кража. Регуляторная ответственность возникает при умышленном несоответствии.

## Заключение

Объёмно-сортовой и поэкземплярный учёт — два разных уровня прослеживаемости маркированных товаров. Для продовольственных категорий чаще применяется ОСУ, для непродовольственных — поэкземплярный. В обоих случаях при продаже через кассу с разрешительным режимом сканируется каждая единица.
