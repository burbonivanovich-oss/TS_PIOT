Центр развития перспективных технологий — оператор национальной системы
цифровой маркировки «Честный знак», определённый Правительством РФ. Генерирует
и выдаёт коды маркировки участникам оборота, ведёт реестр товаров и операций,
администрирует личные кабинеты и API для интеграции. За каждый код маркировки
ЦРПТ взимает плату (стандартно 50 копеек с НДС за код) — за исключением
некоторых категорий по государственным программам. ЦРПТ также разрабатывает
ТС ПИоТ (программный модуль для кассы) и методологические рекомендации по
подключению.
