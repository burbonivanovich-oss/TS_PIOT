Вывод из оборота — завершающая операция жизненного цикла кода маркировки в
«Честном знаке». Основной способ — продажа через онлайн-кассу: при пробитии
чека код маркировки передаётся в систему через ОФД/ТС ПИоТ и автоматически
гасится. Альтернативные причины вывода: утилизация, экспорт, уничтожение брака,
возврат по истечении срока годности. После вывода код не может быть использован
повторно. Нарушение порядка вывода (продажа без фиксации) — основание для
штрафа по ст. 15.12 КоАП РФ.
