Что такое ТС ПИоТ

ТС ПИоТ (техническое средство получения информации о товаре) — программный модуль, интегрируемый в онлайн-кассу или фронтальную систему. В момент продажи он отправляет запрос в информационную систему «Честный знак» и получает разрешение либо запрет на реализацию конкретного экземпляра товара.

Нормативная основа — Федеральный закон от 29.12.2017 № 381-ФЗ с изменениями, внесёнными Федеральным законом от 26.07.2024 № 206-ФЗ.

Как это работает

При сканировании кода маркировки модуль ТС ПИоТ передаёт идентификатор товара оператору — ФГИС МДЛП или ФГИС «Честный знак» в зависимости от категории. Система проверяет статус кода: введён в оборот, выведен, похищен или просрочен.

Если статус кода не позволяет продажу, касса обязана заблокировать операцию и вывести сообщение об ошибке. Это называется «разрешительный режим».

Шаги при сканировании

  1. Кассир сканирует код маркировки на упаковке.
  2. Касса через ТС ПИоТ отправляет запрос в «Честный знак».
  3. «Честный знак» возвращает статус: «Продажа разрешена» или «Продажа запрещена».
  4. При запрете — касса блокирует позицию, кассир сообщает покупателю.

Штрафы за нарушения

Работа без ТС ПИоТ при наличии обязанности приравнивается к неприменению ККТ. По ст. 14.5 КоАП РФ:

  • Должностное лицо — от 25% до 50% от суммы расчёта, но не менее 10 000 ₽.
  • Организация — от 75% до 100% от суммы расчёта, но не менее 30 000 ₽.

При повторном нарушении должностное лицо могут дисквалифицировать на срок до двух лет. Организация рискует приостановкой деятельности до 90 суток.

Часто задаваемые вопросы

01Нужно ли платить за подключение ТС ПИоТ?↓

Само ПО ТС ПИоТ предоставляется «Честным знаком» бесплатно. За обновление прошивки производитель кассы может взять плату — уточняйте у своего поставщика.

02Что делать, если интернет на кассе пропадёт?↓

При отсутствии соединения касса переходит в автономный режим: продажи маркированных товаров допустимы не более 30 дней подряд. Транзакции передаются после восстановления связи.

03Распространяется ли ТС ПИоТ на самозанятых?↓

Нет. Самозанятые не обязаны применять ККТ, поэтому требование по ТС ПИоТ на них не распространяется.

Архитектура CPA-врезки

Данные баннера хранятся в одном месте и не дублируются в каждой статье.

Структура файлов

src/
├── data/
│   └── cpa-banners.ts        ← единственный источник истины
└── components/
    └── CpaBanner.astro       ← рендерит баннер по id

src/data/cpa-banners.ts

Каждый баннер — объект с визуалом, текстами и CTA. Добавить новый = добавить запись в массив.

export interface CpaBanner {
  id:          string;
  eyebrow:     string;   // «Честный знак», «Партнёр» …
  title:       string;   // заголовок (Bebas Neue)
  description: string;   // 1–2 предложения
  cta:         string;   // текст кнопки
  ctaHref:     string;   // URL (affiliate / partner)
  visual: {
    abbrev: string;      // 2 буквы для SVG-заглушки
    bg:     string;      // '#111' | '#E8175D' | путь к картинке
  };
}

Дефолт по категории

Если в frontmatter не указан cpa — берётся дефолт по категории статьи.

// src/layouts/BlogPost.astro
import { CPA_BANNERS } from '../data/cpa-banners';

const CATEGORY_DEFAULT_CPA: Record<string, string> = {
  'ts-piot':         'chestny-znak',
  'markirovka':      'chestny-znak',
  'zakonodatelstvo': 'default-ts-piot',
};

const cpaBannerId = cpa
  ?? CATEGORY_DEFAULT_CPA[primaryCategory ?? '']
  ?? 'default-ts-piot';

const cpaBanner = CPA_BANNERS[cpaBannerId];
