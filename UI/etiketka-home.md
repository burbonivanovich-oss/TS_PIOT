# Главная страница (etiketka-media.ru/) — CPA-аудит

## Оценка: 3/10

## CPA-потенциал
- **Тип трафика:** холодный и тёплый — SEO-органика по информационным запросам («ТС ПИоТ», «маркировка Честный знак», «штрафы ИП»), часть прямых визитов постоянных читателей
- **Целевое действие:** переход на статью → прогрев → клик по партнёрской ссылке/баннеру внутри статьи; дополнительно — прямой переход на партнёра с главной через нативный блок
- **Текущий CPA-статус:** партнёрских ссылок, баннеров, нативных блоков и виджетов нет совсем. Страница не монетизирована

---

## Что работает

1. **Заголовок-крюк.** «РАЗБИРАЕМ ТРЕБОВАНИЯ, КОТОРЫЕ ДОГОНЯЮТ БИЗНЕС» прямо называет боль целевой аудитории — регуляторное давление. Работает на 5-секундном тесте.
2. **Подзаголовок конкретен.** «ТС ПИоТ для онлайн-касс, маркировка «Честный знак», налоги, ЭДО и трудовое законодательство — понятно и без паники.» — сразу сегментирует аудиторию и снижает тревогу («без паники»).
3. **Отличная техническая база.** LCP 360 мс, CLS 0.004 — страница не теряет конверсию из-за производительности. Astro SSG обеспечивает мгновенный first paint.
4. **Тематические тайлы разделов** с счётчиком материалов помогают ориентироваться и ведут на сегментированные разделы.
5. **Свежий контент — 7 статей мая 2026.** Темы высокорелевантны: МРОТ, ЭДО+УПД для маркировки, ЕНС, Календарь маркировки. Это прогретый трафик.

## CPA-проблемы (приоритизированные)

### #1 — критично: На главной нет ни одного партнёрского элемента

**Описание:** Вся страница — навигационный хаб. Ни один элемент не ведёт к партнёру: ни ОФД, ни кассы, ни ЭДО, ни онлайн-бухгалтерия. Пользователь, который пришёл на главную и не провалился в статью, уходит без единого контакта с монетизацией. При допущении, что 30–40 % визитов на главную не переходят дальше — это прямые потери CPA.

**Рекомендация:** Добавить между секцией разделов (// 01) и лентой материалов (// 02) блок «Сервисы для вашего бизнеса»: 2–3 партнёрские карточки (ОФД, ЭДО, онлайн-касса). Каждая карточка — лого партнёра, одна строка выгоды, кнопка «Подключить» с UTM-ссылкой.

**Усилие:** M

---

### #2 — критично: CTA «Все статьи» никуда не конвертирует в CPA

**Описание:** Главная черная кнопка `btn btn-primary` с текстом «Все статьи» ведёт на `/blog/` — ленту материалов. Это правильный редакционный путь, но для CPA-монетизации он холодный: на странице блога нет партнёрских блоков (по данным HTML). Ghost-кнопка «Что такое ТС ПИоТ →» ведёт в `/category/ts-piot/` — тематический раздел без монетизации.

**Рекомендация:** Добавить третий CTA с конкретным партнёрским направлением: например, «Подобрать ОФД для ТС ПИоТ →» рядом с ghost-кнопкой. Или изменить ghost-кнопку на «Как подключить ОФД →» с ссылкой на партнёрскую посадочную. Основной CTA «Все статьи» оставить без изменений — он нужен для SEO-глубины.

**Усилие:** S (замена href и текста одной кнопки)

---

### #3 — высокий: Инструменты (Калькулятор штрафов, Календарь) не анонсированы на главной

**Описание:** Калькулятор штрафов и Календарь маркировки — страницы с максимальным intent: пользователь уже думает о конкретной проблеме. Это идеальные точки для CPA-размещения. Но с главной они видны только в футере в колонке «Инструменты» — ниже линии прокрутки для большинства пользователей.

**Рекомендация:** Добавить четвёртый тайл в секцию разделов (рядом с ТС ПИоТ, Маркировкой, Законодательством): «Инструменты» с подписью «Калькулятор штрафов, Календарь маркировки». Или вынести их отдельным блоком перед footer. Это увеличит трафик на страницы, где партнёрское размещение наиболее органично.

**Усилие:** S

---

### #4 — высокий: Нет захвата email-адресов

**Описание:** Отсутствует форма подписки. B2B-медиа с регуляторной тематикой — классический сценарий для email: дедлайны меняются, предприниматель хочет не пропустить изменение. Email-база даёт повторные касания с партнёрскими материалами в письмах.

**Рекомендация:** Добавить компактный блок подписки перед футером: «Изменения в маркировке и ККТ — на почту раньше, чем придёт штраф. [поле email] [Подписаться]». После подтверждения — автоматическое письмо с 2–3 нативными рекомендациями партнёров.

**Усилие:** L (нужен ESP, автосерия, форма)

---

### #5 — средний: Лента статей не содержит промежуточных партнёрских блоков

**Описание:** Сетка из 7 карточек `.posts-grid` — чистый контент без единого нативного рекламного блока. Стандарт в контентных медиа: 1 партнёрский блок на 3–4 редакционных материала.

**Рекомендация:** После 3-й карточки статей вставить нативную карточку того же визуального формата (post-card) с пометкой «Партнёр» и текстом, например: «ОФД для ТС ПИоТ — подключите за 10 минут. [Конкурс Партнёр] →».

**Усилие:** M

---

## Быстрые CPA-победы (реализация менее 1 дня)

1. **Добавить в навигацию ссылку «Калькулятор штрафов»** между «Кассы» и «Маркировка» — прямой вход на страницу с высоким intent, где логично разместить партнёрский ОФД.
2. **Изменить ghost-кнопку «Что такое ТС ПИоТ →»** в hero на «ОФД для ТС ПИоТ →» со ссылкой на статью `/blog/2026-05-01-ofd-dlya-ts-piot/` — там проще всего начать монетизацию с первого партнёрского блока.
3. **В футере в колонке «Инструменты» добавить строку** «Подобрать ОФД →» с партнёрской UTM — это единственная зона, где сейчас уже есть структурированные ссылки, расширить за минуты.
