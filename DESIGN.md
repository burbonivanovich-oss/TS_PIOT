## Концепция

**Современный деловой медиа** — понятно, ярко, без занудства.
Ориентир: Тинькофф Журнал, Morning Brew. Аудитория — предприниматель
или бухгалтер, которому нужен быстрый ответ. Никакой «бухгалтерской серости».

## Типографика

**Заголовки:** Commissioner Bold — строгий, чёткий.
**Тело:** Geologica — современный, читаемый.

### Размеры

| Токен             | rem      | Применение         |
|-------------------|----------|--------------------|
| `--text-display`  | 2.5rem   | Hero-заголовок     |
| `--text-h1`       | 2rem     | Заголовок статьи   |
| `--text-h2`       | 1.5rem   | Раздел H2          |
| `--text-h3`       | 1.2rem   | Подраздел H3       |
| `--text-body`     | 1.125rem | Тело (18px)        |
| `--text-ui`       | 1rem     | Навигация, кнопки  |
| `--text-sm`       | 0.875rem | Мета, теги         |
| `--text-xs`       | 0.8125rem| Caption            |

Line-height тела: **1.75**.
Заголовки: **1.2**.

## Пространство

8px-сетка.

```
--sp-1:  4px    --sp-5: 24px
--sp-2:  8px    --sp-6: 32px
--sp-3: 12px    --sp-7: 48px
--sp-4: 16px    --sp-8: 64px
```

**Радиусы** — дружелюбные, не угловатые:

```
--r-sm:  6px   (кнопки, поля)
--r-md:  10px  (карточки)
--r-lg:  14px  (hero, большие блоки)
--r-pill: 999px (бейджи, теги)
```

## Ключевые компоненты

### Header
- Белый фон, нижняя граница `--border`.
- Лого: Commissioner Bold, `--text`, крупно.
- Цветная точка / полоска рядом с лого — `--accent`.
- Навигация компактная, active-ссылка — `--accent`.

### Карточка статьи
- Белый фон, рамка `--border`, radius `--r-md`.
- Hover: box-shadow, рамка `--accent`.
- Категория — цветной badge сверху.

### Callout
- Левая полоса 4px + цветной фон по типу.
- Radius `--r-md` справа.

### Теги
- Pill-форма (`--r-pill`).
- Фон `--bg-muted`, цвет `--text-3`.
- Hover: фон `--accent-light`, цвет `--accent`.

### Кнопки
- Primary: `--accent`, белый текст, radius `--r-sm`.
- Ghost: прозрачный, граница `--accent`.
- Padding: `0.6em 1.25em`.

### Footer
- Тёмный фон `#0F172A`.
- 4 колонки, ссылки белые/серые.
