# Дизайн-система «Этикетка»

Версия 1.0 · 2026

---

## Концепция

**«Деловая газета онлайн»** — строгое редакционное издание для предпринимателей.
Не стартап, не госпортал. Ближайшие эстетические ориентиры: Коммерсантъ, WIRED,
Bloomberg Businessweek. Ключевые качества: доверие, плотность информации,
читаемость, профессиональный голос.

Пользователь — занятой предприниматель или бухгалтер, которому нужен ответ,
а не дизайн-впечатление. Интерфейс отступает, контент выходит вперёд.

## Типографика

### Стек шрифтов

| Роль          | Семейство                        | Начертания |
|---------------|----------------------------------|------------|
| Заголовки     | Commissioner → Inter → sans      | 400, 700   |
| Тело текста   | Geologica → Inter → sans         | 400, 700   |
| UI-мелочи     | Inter → system-ui → sans         | 400, 600   |

Шрифты — самохостинг в `public/fonts/`. Georgica несёт основной читательский
опыт, Commissioner создаёт контраст редакционного заголовка.

### Типографическая шкала

| Токен           | rem    | px (base 16) | Применение              |
|-----------------|--------|--------------|-------------------------|
| `--text-display`| 3rem   | 48px         | Лендинг-герой           |
| `--text-h1`     | 2.25rem| 36px         | Заголовок статьи        |
| `--text-h2`     | 1.625rem| 26px        | Раздел H2               |
| `--text-h3`     | 1.25rem| 20px         | Подраздел H3            |
| `--text-h4`     | 1.0625rem| 17px        | H4, lead-подписи        |
| `--text-body`   | 1.1875rem| 19px        | Тело статьи             |
| `--text-ui`     | 1rem   | 16px         | Навигация, кнопки       |
| `--text-sm`     | 0.875rem| 14px        | Мета, дата, теги        |
| `--text-xs`     | 0.8125rem| 13px       | Caption, breadcrumbs    |

### Межстрочный интервал

| Контекст        | line-height |
|-----------------|-------------|
| Display / H1    | 1.15        |
| H2 / H3         | 1.25        |
| Тело статьи     | 1.78        |
| UI / meta       | 1.4         |

### Трекинг заголовков

| Уровень   | letter-spacing |
|-----------|----------------|
| H1        | -0.03em        |
| H2        | -0.02em        |
| H3        | -0.015em       |

Русскоязычный текст в гротеске выглядит плотнее латиницы — компрессия
letter-spacing компенсирует и добавляет газетной строгости.

4px-сетка. Базовый ритм — 4px, стандартный блок — 8px.

| `--sp-4`     | 16px     | Стандарт             |

## Компоненты

### Header (masthead)

Sticky. Белый фон с нижней границей `--base-150`.
- Логотип: Commissioner Bold, `--ink`, текст «Этикетка» — без иконки-квадрата.
  Рядом — тонкая полоска акцента слева (3px solid `--accent`).
- Навигация: `--text-ui`, цвет `--base-700`. Active: `--ink`, нижняя линия `--accent`.
- Mobile (≤820px): бургер, выпадающий список.

### Article layout

- Ширина колонки: 700px max-width, центрирована.
- Фон: `--paper` для всего экрана; контент на нём без дополнительного фона.
- Breadcrumbs: `--text-xs`, `--base-500`.
- H1: Commissioner Bold, `--ink`, `--text-h1`, tracking -0.03em.
- Лид (description): Geologica, `--text-h4`, `--base-600`, line-height 1.55.
- Мета (дата, время чтения): `--text-sm`, `--base-600`.

### Blog card

Вертикальная карточка для сеток:
- Фон `--surface`, рамка `1px solid --base-200`, radius `--r-sm`.
- Hover: рамка `--accent`, тень слабая `0 2px 12px rgba(16,15,15,0.06)`.
- Категория: цветной бейдж, radius `--r-xs`, padding 2px 6px.
- Заголовок H3: Commissioner, `--ink`, `--text-h3`.
- Описание: Geologica, `--text-sm`, `--base-600`.
- Дата: `--text-xs`, `--base-500`.

### Callout

Три типа. Левая полоса 3px. Без иконки — текст должен говорить сам.

```
.callout          → border --accent   / bg --accent-soft
.callout-warn     → border --warn-fg  / bg --warn-bg
.callout-danger   → border --danger-fg/ bg --danger-bg
```

Радиус 0 справа, `--r-sm` слева.

### Blockquote

Только левая полоса 3px `--base-300`. Без фона. Текст `--base-700`, курсив.
Газетная цитата — никаких декоративных кавычек.

### Tags

Прямоугольные (`--r-xs`). Фон `--base-100`, цвет `--base-700`.
Hover: фон `--accent-soft`, цвет `--accent`.
Размер: `--text-xs`, padding 3px 8px.

### TOC (таблица содержания)

`<details>`. Левая полоса `2px solid --base-150`.
Фон `--base-100`, radius `--r-md`. Шрифт `--text-sm`, `--base-700`.
Без стрелки-треугольника — summary с текстом «→ Содержание».

### Progress bar (полоса прокрутки)

2px высота, `--accent`, fixed top.

### Кнопки

```
.btn-primary  → bg --accent,   color #fff,    radius --r-sm
.btn-ghost    → bg transparent, border --accent, color --accent
```

Hover: primary → `--accent-hover`; ghost → bg `--accent-soft`.
Padding: 0.65em 1.2em. Font: `--text-ui`, weight 600.

### Share bar

Без рамки-карточки. Просто строка: «Поделиться:» + ссылки-кнопки.
Telegram: фон `#E8F4FD`, цвет `#0088CC`.
ВКонтакте: фон `#E8F0FB`, цвет `#2787F5`.

### Related posts

Заголовок «Читайте также» — H2, Commissioner.
Сетка 3 карточки. Категория — цветной бейдж.

### Footer

Фон `--ink` (`#100F0F`), текст `rgba(255,255,255,0.65)`.
4 колонки. Разделитель `rgba(255,255,255,0.08)`.
Копирайт: `--text-sm`.

## Категорийные цвета в UI

Компонент бейджа категории использует data-атрибут:

```html
<span class="cat-badge" data-cat="ts-piot">ТС ПИоТ</span>
<span class="cat-badge" data-cat="markirovka">Маркировка</span>
<span class="cat-badge" data-cat="zakonodatelstvo">Законодательство</span>
```

```css
.cat-badge[data-cat="ts-piot"]         { color: var(--accent);       background: var(--accent-soft); }
.cat-badge[data-cat="markirovka"]      { color: var(--cat-markirovka); background: var(--cat-markirovka-soft); }
.cat-badge[data-cat="zakonodatelstvo"] { color: var(--cat-zakon);     background: var(--cat-zakon-soft); }
```

## Чего избегать

- Тени крупнее `0 2px 12px rgba(16,15,15,0.08)` — выглядят как SaaS, не как издание.
- Border-radius > 6px на блочных элементах.
- Синий фон на больших областях (только акцент, не заливка).
- Жирный текст в теле статьи без реальной нужды.
- Анимации дольше 200ms.
- Эмодзи в интерфейсе.
